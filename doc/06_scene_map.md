# 06 関係者・シーンマップ

ライターエージェントが「シーンが浮かぶ投稿」を作るための参照ファイル。
Sは全ての関係者の中心にいる。

投稿を作る時は必ず「誰との・どのシーンか」を特定してから書くこと。

---

## 社内：上方向

### 上司

```
属性：管理職寄り。制作出身だが今は数字と調整が仕事。
性格：アウトプットと納期で判断。「それってどう売上につながるの？」が口癖系。
力関係：Sが下だが、現場判断はSに委ねられてる。
温度感：尊重してるけど、クリエイティブの話は噛み合わない時がある。

やりとりの特徴：
├── 企画の背景より「クライアントは納得してるか」を先に聞く
├── AIの話をすると「コスト削減できるの？」に変換される
└── Sが根回しして通す場面が多い

シーンの例：
「上司にAIカンプを見せたら最初に出た言葉が
 『これ安くできる？』だった。
 クリエイティブより先にコストの話になる現実。」
```

### ボス（社長）

```
属性：創業者か近い存在。感覚で動く。
性格：ビジョンと直感で判断。「なんかいい感じか」で決める。会議より廊下で決まる人。
力関係：最終決裁者。「社長の直感をどう形にするか」がSの仕事の一部。
温度感：距離は近いが読めない。機嫌と文脈で全部変わる。

やりとりの特徴：
├── 「なんかもっとこう、ドーンとしたやつ」みたいな指示が来る
├── 前の会議の決定が翌日覆る
└── 刺さった時は「これでいこう」が早い

シーンの例：
「社長から『もっとドーンとしたやつで』って言われて
 AIに投げたら、意外と社長の感覚に近いものが出てきた。
 人間より社長の直感を理解してるかもしれない。笑」
```

## 社内：横方向

### 同僚クリエイティブディレクター

```
属性：同じ立場だが、得意領域・スタイルが違う。
性格：
  ①競争型：「俺の方がいいもの作る」が原動力
  ②協調型：「一緒にいいもの作ろう」で動く
力関係：横並びだが、案件の規模や社内評価で微妙な上下が生まれる。
温度感：戦友でもあり、ライバルでもある。本音が言えるのはこの人たち。

やりとりの特徴：
├── 「お前、これどうやって作ったの？」が頻繁に来るようになった
├── AIを使ってることを言うべきか迷う空気がある
└── 「AIで作ったって言ったらどう思う？」を探り合ってる

シーンの例：
「同じCDの同僚に、AIで作ったカンプを
 何も言わずに見せたんだけど。
 『これ外注？誰に頼んだの？』って聞かれた。
 まだ言ってない。笑」
```

### 部下（デザイナー・スタッフ）

```
属性：若手〜中堅。デジタルネイティブ世代。
性格：
  ①積極型：自分から使おうとしてる・ツール好き
  ②不安型：「自分の仕事がなくなるんじゃ」と思ってる
力関係：Sが上。でも「下の者ができない原因は上にある」が信念。
温度感：教えたい、でも過保護にはしたくない。

やりとりの特徴：
├── AIの使い方を聞いてくる頻度が増えてきた
├── 指示の出し方がAI前後で変わってきた
└── 部下がAIを使い始めたら、Sが求めるもののレベルが上がった

シーンの例：
「部下がAI使い始めてからラフの量が3倍になった。
 量は増えたけど、どれを選ぶか考える時間は増えた。
 楽になってるのかなんなのか、まだわかんない。笑」
```

## 外部：代理店

### 広告代理店営業（ソルジャー系）

```
属性：20〜30代前半。数字と納期で動く。
性格：とにかく早く・安く・無茶振りが仕事。悪気はない、ただ現場が見えていない。
      「なんとかなりますよね？」が口癖。
力関係：発注者側だが、現場力はSの方が上。
温度感：憎めないけど毎回しんどい。

やりとりの特徴：
├── 「明日までに3パターン」が普通に来る
├── 修正は「ちょっとだけ」が実は大幅変更
└── AIを知らないので「なんで早いの？」と不思議がる

シーンの例：
「昨日、代理店から夕方5時に
 『明日の朝イチで3パターンほしい』って連絡。
 以前の自分なら確実に徹夜だった。
 感謝はしてないけど。笑」
```

### 広告代理店営業（部長クラス）

```
属性：40代前後。戦略と関係構築が仕事。
性格：数字も見るが、長期の関係性と信頼で動く。
      「一緒にいいものを作りたい」が本音。ただし社内の上に弱い。
力関係：Sと対等に近い。クリエイティブを尊重してくれる。
温度感：話が通じる。本音が言える数少ない代理店側の人間。

やりとりの特徴：
├── 戦略の話ができる
├── 「AIをどう使ってるか教えてほしい」と聞いてくる
└── 「一緒にAI活用の提案を持ち込もう」と言ってくる

シーンの例：
「代理店の部長から
 『うちのクライアントにAI活用の提案したいんだけど
 一緒に考えてもらえる？』って連絡。
 一年前は『AIって大丈夫なの？』って顔してた人が。
 時代が動いてる。」
```

## 外部：クライアント

### 直案件クライアント（担当者レベル）

```
属性：事業会社のマーケや広報担当。
性格：社内の調整に追われてる。「上に説明できるか」が判断基準。
      AIに興味はあるが、リスクが怖い。
力関係：発注者。でもクリエイティブ判断はSに委ねてる。
温度感：信頼関係ができてる案件は本音が言い合える。

やりとりの特徴：
├── 「AIで作ったって社内でバレたらまずいですか？」と聞いてくる
├── 修正の方向性がふわっとしてる（「なんか違う」を言語化できない）
└── AIカンプを見せると「もうこれでいいじゃないですか」が増えた

シーンの例：
「クライアントの担当者から
 『これ、AIで作ったって上に言っていいですか？』って聞かれた。
 言っていいかどうかより
 『いいものかどうか』で判断してほしいんだけど。
 社内政治があるのはわかってる。難しいところだよな。」
```

### 直案件クライアント（決裁者・役員クラス）

```
属性：50代前後が多い。最終Go/No goを出す人。
性格：「ブランドとして正しいか」「リスクがないか」で判断。
      AIに懐疑的か、過剰に期待するかの二極。
力関係：最終決裁者。この人がNOと言ったら全部やり直し。
温度感：距離がある。Sが直接話せる案件は限られる。

やりとりの特徴：
├── 「AIってうちのブランドイメージ大丈夫なの？」
├── 「これ、本物の写真じゃないとダメだよね？」
└── 逆に「全部AIでやってコスト半分にできない？」

シーンの例：
「クライアントの役員から
 『AIで作ったってわかったら消費者に嘘をついてることにならないか？』
 って聞かれた。
 写真だって昔からレタッチしてるし、どこからが嘘なのかって話で。
 答えはまだ出してない。でも考えてる。」
```

## 外部：ブレーン・専門職

### 外部コピーライター

```
属性：言葉のプロ。フリーか小さい事務所所属が多い。
性格：言葉にこだわりがある。AIのコピーに一番センシティブ。
力関係：対等。Sはビジュアル、CWは言葉、という役割分担。
温度感：クリエイティブの話が一番深くできる相手。

やりとりの特徴：
├── AIのコピーに対して一番率直な批評をしてくれる
├── 「これ、AIっぽい」という感覚が一番鋭い
└── 一緒にブレストするとAIが第三の視点になってる

シーンの例：
「外部のCWと久しぶりにブレストしたんだけど、
 途中でClaudeを入れてみた。
 CWが『この方向は違う』って言って
 Claudeが『この方向もある』って出してきて
 僕が両方を見ながら第三案を考える。
 AIが来てからブレストの構造が変わった気がする。」
```

### カメラマン

```
属性：現場のプロ。映像・スチール両方いる。
性格：技術への誇りがある。AIへの反応は「驚き」「脅威」「興味」が混在。
力関係：専門分野では向こうが上。でも案件全体の方向性はSが決める。
温度感：現場でのやりとりが多い。リアクションが率直。

やりとりの特徴：
├── AIカンプを見せた時のリアクションが一番素直に出る
├── 「これAIですか？」という質問が増えてきた
└── 「AIでできるなら撮影いらないじゃないですか」という声も

シーンの例：
「撮影前にAIで作ったイメージカンプ、
 カメラマンに見せてみたんだけど。
 『これ、どこで撮ったんですか？』って聞かれた。笑
 AI生成ですって言ったら
 『え、これでいいじゃないですか』ってなって
 ロケ地の候補が一個減った。」
```

### エンジニア（フロントエンド・バックエンド・AIエンジニア）

```
属性：開発のプロ。クリエイターとは別の視点でAIを見てる。
性格：技術的な正確さを重視。クリエイターのAI活用に興味がある人も多い。
力関係：開発領域では向こうが上。クリエイティブ領域はSが上。
温度感：ビジネスライクだが、AI × クリエイティブの話では熱くなることも。

やりとりの特徴：
├── 「クリエイターがAIをどう使ってるか」を聞いてくる
├── 「それ、技術的にはこういう仕組みで」と補足してくれる
└── お互いの領域をAIで補完し合えることがわかってきた

シーンの例：
「AIエンジニアと話してたら
 『クリエイターがAIを使いこなすのが
 一番面白い使い方だと思う』って言われた。
 嬉しいような、でもまだそれを証明できてない感覚もある。」
```

## シーン生成の3要素（ライターへの指示）

```
投稿を作る時は必ずこの3要素を特定してから書く：

① 誰と
（上司・部下・代理店ソルジャー・クライアント担当・カメラマン...）

② どんな状況で
（提案前日・修正対応・ブレスト・納品直前・撮影前...）

③ AIがどう介在したか
（たたき台を作った・時間が半分になった・説明に使った...）

この3要素が揃うと「シーンが浮かぶ投稿」になる。
答えられない投稿は現場感がない＝書き直し。
```

```

1週間の投稿に最低4種類の関係者が登場する
「まだ登場していない関係者」のアラートを毎週確認する
```
